# Image Cropper App

## Description

This web application allows users to:

- Upload JPG or PNG images
- Crop images using different aspect ratios
  - 1:1
  - 4:3
  - 16:9
  - Free ratio
- Download the cropped image

## Technologies Used

- HTML
- CSS
- JavaScript
- Cropper.js

## GitHub Pages
